NUMBER SYSTEM

Decimal/Binary/Octal/Hexadecimal Number System & Conversion

In computing, different number systems are used to represent data.

  • Decimal (Base-10): The standard system we use daily (0-9).
  • Binary (Base-2): Used in digital electronics (0 and 1).
  • Octal (Base-8) and Hexadecimal (Base-16): Compact ways to represent binary numbers.

This section covers how to convert between these systems.

Binary Calculations

Binary arithmetic is crucial in computer operations. This section includes:

  • Binary Addition: Similar to decimal addition, with carrying over when summing 1 + 1.
  • Binary Subtraction: Uses borrowing, often employing two’s complement.
  • Binary Multiplication and Division: Similar to decimal methods but using only 0s and 1s.

One’s and Two’s Complement Method for Binary Subtraction

When subtracting binary numbers, computers use complements:

  • One’s Complement: Inverting all bits (0s to 1s, 1s to 0s).
  • Two’s Complement: Adding 1 to the one’s complement, making subtraction easier.

This method helps computers handle negative numbers efficiently.

Logic Function & Boolean Algebra

Introduction to Boolean Algebra

Boolean algebra is a mathematical system for logic operations. It forms the foundation of digital circuits.

  • Uses binary values (0 and 1) to represent logical states.
  • Involves operations like AND, OR, and NOT.

Introduction to Boolean Values, Truth Table, Boolean Expression, and Boolean Function

  • Boolean Values: 0 (False) and 1 (True).
  • Truth Table: A table showing the output of logic operations for all possible inputs.
  • Boolean Expressions: Equations that define logic circuits (e.g., A + B for OR, A · B for AND).
  • Boolean Functions: Functions that use Boolean expressions to represent digital logic.

Logic Gate

Logic gates are electronic devices implementing Boolean functions.

  • Basic Gates: AND, OR, NOT.
  • Derived Gates: NAND, NOR, XOR, XNOR.

Gates process binary inputs to produce outputs, forming the building blocks of digital circuits.

Laws of Boolean Algebra

Boolean algebra simplifies logic circuits using laws like:

  • Identity Law: A + 0 = A, A · 1 = A
  • Domination Law: A + 1 = 1, A · 0 = 0
  • De Morgan’s Theorems: Helps simplify NOT operations in complex expressions.

Statement and Verification of Laws of Boolean Algebra using Truth Table

Each Boolean law can be proven using a truth table by evaluating outputs for all input combinations.
