CHAPTER-3
COMPUTER SOFTWARE AND OPERATING SYSTEM

Concept of Software

Definition of Software

Software refers to a collection of programs and data that tell the computer how to operate. Unlike hardware, software is intangible and provides instructions for the system to execute various tasks.

Categories of Software

  • System Software: Includes operating systems (Windows, Linux, macOS) and utility programs that help manage computer hardware.
  • Application Software: Programs designed for end users, such as word processors, web browsers, and games.
  • Utility Software: Tools that optimize system performance, such as antivirus programs, disk cleaners, and backup utilities.

Concept of Operating System

Introduction to Operating System

The OS acts as a bridge between users and hardware. It controls essential functions like memory management, process scheduling, and file handling.

Role of Operating System

  • Resource Management: Manages CPU, memory, storage, and peripherals.
  • Process Management: Handles multiple running applications (multitasking).
  • User Interface: Provides interaction methods like Command Line Interface (CLI) and Graphical User Interface (GUI).
  • Security and Access Control: Protects data through authentication and encryption.

Functions of an Operating System

  • Memory Management: Allocates memory to running applications.
  • File System Management: Organizes and manages stored files and directories.
  • Device Management: Controls input/output devices like printers, keyboards, and USB drives.
  • Error Handling: Detects and responds to system failures.

Windows Operating System

Introduction to GUI-based OS and its Features

Windows provides a Graphical User Interface (GUI) with icons, windows, and menus for easy navigation.

Managing Files and Folders with File Explorer

File Explorer helps users organize, search, and manage files.

Open Source and Mobile Operating Systems

Concept of Open Source OS

Open-source operating systems like Linux allow developers to modify and distribute the software freely.

Introduction to UNIX, Linux

  • UNIX: A powerful, multi-user OS used in servers and workstations.
  • Linux: A UNIX-based OS available in different distributions (Ubuntu, Fedora, Debian).

Concept of Mobile Operating System

Mobile OS powers smartphones, tablets, and wearable devices. Examples: Android (Google), iOS (Apple), Windows Mobile.
